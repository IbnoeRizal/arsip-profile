* **Struktur folder**

```
    |- app/|- api/ (api endpoint)
    |      |- karyawan/ frontend (untuk public,user,admin)
    |      |- (admin)/  frontend (untuk admin)
    |
    |- components/ (fragmentasi komponen ui untuk frontend)
    |- context/ (group react context untuk frontend)
    |
    |
    |- lib/ (core logika backend)
    |
    |- prisma/|- factory/ (data dummy generator)
    |         |- migration/ (db migration generated by prisma)
    |
```


## Untuk memulai pengembangan aplikasi
**siapkan** :
1. `git` version control system
2. RDBMS (relational database management system) disarankan `postgresql`
3. `npm` dan `node`

---

**Duplikasi Proyek ke Computer Lokal (jalankan di terminal)**
```pwsh
    git clone https://github.com/IbnoeRizal/arsip-profile.git
```

**Jalankan di terminal**
```pwsh
    cp .env.example .env
```

**Modifikasi file .env**
```env
    DATABASE_URL="isi dengan string koneksi ke database yang digunakan"

    #contoh
    #DATABASE_URL="postgresql://postgres:(password_local)@localhost:5432/(nama_database)?schema=public"

    JWT_SECRET="token yang ingin digunakan (bebas)"
```

**Install dependensi (jalankan di terminal)**
```pwsh
    npm install
```

## Menjalankan Proyek

**Jalankan di terminal**
```pwsh
    npm run dev
```
